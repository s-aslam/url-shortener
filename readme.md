# REST API Node Server

## Description

1. install node js from [https://nodejs.org/en/download/]

## Getting Started

### Installation

Install the dependencies:

```bash
npm install
```

Run:
```bash
npm start
```

## Project Structure

```
 |--models          # Mongoose models (data layer)
 |--utils           # Utility classes and functions
 |--app.js          # Express app & App entry point
```
